Descripcion del proyecto

Este proyecto consiste en la creación de un portafolio web personal diseñado para mostrar de forma profesional la información académica, habilidades, proyectos y contacto de un desarrollador o profesional. Desarrollado con HTML, CSS y Git, el portafolio incluye una sección "Sobre mí" con biografía y objetivos, "Educación" con la formación académica y cursos relevantes, "Habilidades" destacando las tecnologías y destrezas adquiridas, "Proyectos" donde se enlaza el repositorio de GitHub y se incluye una captura del sitio, y "Contacto" con enlaces a redes profesionales. El diseño visual emplea un esquema de colores que combina morado y crema, brindando una estética moderna y profesional.
